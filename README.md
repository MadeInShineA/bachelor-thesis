## Local Setup

To set up the project website locally, clone the repository and either:

### Using Nix with Flakes enabled (recommended)

```bash
nix develop
quarto render
```

### Without Nix

1. Install [Quarto](https://quarto.org/docs/get-started/)
2. Run `quarto render`

The site will be in the `_site/` directory.

## Submodules

This repository uses Git submodules to include related projects:

- **bachelor-thesis-project**: Contains the project implementation, including Python scripts, Jupyter notebooks, and associated resources.
- **bachelor-thesis-report**: Contains the bachelor thesis report written in [Typst](https://typst.app/), along with figures, bibliography, and the compiled PDF.

To initialize and update submodules after cloning:

```bash
git submodule init
git submodule update
```
